# 🔍 Exploratory Data Analysis — Titanic Dataset

A full EDA project using Python on *Google Cloud Vertex AI Workbench*. The dataset contains real passenger data from the 1912 Titanic disaster.

---

## 🛠️ Tools & Libraries
- Python (Pandas, NumPy, Matplotlib, Seaborn)
- Google Cloud Vertex AI Workbench
- JupyterLab

---

## 📋 What I Did
- Loaded and explored the Titanic dataset (891 rows, 15 columns)
- Found and handled missing values (age, embarked, deck)
- Removed 116 duplicate rows
- Created visualisations to find patterns in survival rates
- Engineered new features (family size, age groups)
- Encoded categorical variables to prepare data for ML modelling

---

## 📊 Key Findings
- Only *41.3%* of passengers survived
- *Women* had a much higher survival rate than men
- *1st class* passengers survived significantly more than 3rd class
- The dataset was cleaned from *891 → 775 rows* and *15 → 20 columns* after processing
